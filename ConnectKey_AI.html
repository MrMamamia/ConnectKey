System Prompt (Per-chat)

Keys live only in memory (lost on refresh). Chat history (text/attachments) is saved locally in IndexedDB.

Web Search keys

Firecrawl is tried first for the "Web Search" toggle below the message box. If it errors, times out, or its key is empty, Tavily is used instead. These two are saved locally (unlike endpoint keys, which stay memory-only) so you don't have to re-paste them each session.

Customize

Saved locally. Applied to every chat as context, on top of any per-chat system prompt.

Appearance

Saved locally, applies every time you open this file.

Backup

Exports/imports chats, current chat, web search keys, appearance prefs, and your profile as a JSON file. Endpoint API keys are session-only and are not included. Importing replaces your current chats and settings.
